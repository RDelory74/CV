Dongeon et Dragon en Java

L'utilisation des classes en Programmation orienté objet

Lien vers le Github

Le projet Donjons & Dragons réalisé en langage Java avec une base de donnée MySQL. Grâce à ce projet, j'ai acquis des compétences avancées en programmation orientée objet pour structurer un code robuste et évolutif. J'ai également appris à gérer des bases de données relationnelles pour stocker les personnages et leurs inventaires. Enfin, cela m'a permis d'aborder l'importance de factorisation du code.

Dans ce jeu, les joueurs peuvent incarner deux types de héros, le mage et le guerrier. Au fil de l'avancée sur le plateau je joueur fera des rencontres aléatoires d'énemis et gagnera en xp et donc en niveau au fil de l'aventure.

Rentakar, la location de véhicule en Micro-Service

L'utilisation de Spring Boot avec Maven et MySQL

Lien vers le Github

Le projet Rentakar est un système de location de véhicules basé sur une architecture de microservices, développé en utilisant Spring Boot, Vue.js, MySQL, et une passerelle API avec Spring Cloud Gateway.

Grâce à ce projet, j'ai approfondi mes compétences en conception et déploiement de microservices, notamment la gestion de la communication entre services via Eureka pour la découverte de services. J'ai également renforcé mes connaissances en intégration frontend-backend, en utilisant Vue.js pour créer une interface utilisateur dynamique et réactive. De plus, ce projet m'a permis de configurer une base de données relationnelle avec MySQL et d'assurer la sécurité et les performances via des pratiques comme la configuration de CORS et la gestion des délais dans les requêtes.

Le système est composé de plusieurs services :

VehiculeRentakar:
Gestion des véhicules, incluant leur disponibilité et leurs spécifications.


UserRentakar:
Gestion des utilisateurs et de leurs données.


GatewayRentakar:
Point d'entrée unique pour les clients, permettant de rediriger les requêtes vers les services appropriés..


OrderRentakar:
Gestion des réservations, permettant de vérifier la disponibilité des véhicules et d'enregistrer les commandes.


Eureka-Server:
Découverte des micro-services et gestion des ports pour les relations.


Chaque service est indépendant mais interconnecté, mettant en pratique cela permet une maintenance plus facile à gérer, il est plus facile de faire évoluer le site aussi.

Spark Tales

PHP en native au service d'une histoire dont vous êtes le héros

Lien vers le Github

Ce projet consiste à créer un système permettant de gérer et d'afficher les contributions des utilisateurs pour enrichir l'histoire d'Avian Spark Tales, un univers narratif collaboratif. Les utilisateurs peuvent soumettre leurs idées via un formulaire, et ces idées sont stockées dans un fichier texte ou une base de données pour être ensuite affichées dynamiquement sur le site web.

Compétences couvertes lors de cet exercice:

Récupération de données utilisateurs.


Stockage dans une structure.


Manipulation et affichage des données.


Une potentielle améloration visant à implémenter une moderation serait un plus.
